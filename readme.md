# 📚frontend-fundamentals-Pweb

Repositório com os projetos e exercícios desenvolvidos na disciplina de **Programação para Web** do curso de Análise e Desenvolvimento de Sistemas — FATEC Sorocaba.

Os projetos evoluem ao longo do semestre, partindo de fundamentos de HTML/CSS até orientação a objetos em JavaScript e desenvolvimento back-end com Node.js.

---

## 🗂️ Projetos

### 🌙 [the-moon-artemis-program](./the-moon-artemis-program)
Página informativa sobre a Lua e o programa Artemis da NASA. Uso de grid layout, tabelas, vídeo embed e estilização temática com gradientes escuros e paleta espacial.

**Tech:** HTML · CSS · CSS Grid · Media Elements

---

### 🃏 [rock-paper-scissors-game](./rock-paper-scissors-game)
Jogo de Pedra, Papel e Tesoura contra o computador. Resultado gerado aleatoriamente com lógica de comparação e feedback visual ao usuário.

**Tech:** HTML · CSS · JavaScript (DOM, Math.random)

---

### ⚖️ [bmi-calculator](./bmi-calculator)
Calculadora de IMC com classificação por faixas (Magreza, Normal, Sobrepeso, Obesidade, Obesidade Grave), feedback visual com cores e animação de erro nos campos inválidos.

**Tech:** HTML · CSS · JavaScript (DOM, validação, animação CSS)

---

### 🏦 [rectangle-and-bank-account-oop](./rectangle-and-bank-account-oop)
Cálculo de área de retângulo e sistema de contas bancárias (corrente e poupança) usando herança entre classes JavaScript. Entrada via formulário com rádio buttons.

**Tech:** HTML · CSS · JavaScript (Classes ES6, herança, encapsulamento)

---

### 📋 [survey-form-with-stats](./survey-form-with-stats)
Formulário de pesquisa de opinião com acumulação de respostas: calcula média de idade, maior/menor idade, porcentagem de opiniões positivas e contagem por gênero.

**Tech:** HTML · CSS · JavaScript (arrays, funções de agregação)

---

### 📝 [form-validation-demo](./form-validation-demo)
Formulário completo com validação de campos (nome, e-mail, comentário, rádio buttons), mensagens de erro inline e exibição dinâmica de feedback para o usuário.

**Tech:** HTML · CSS · JavaScript (validação, eventos de formulário)

---

### 🔡 [js-string-array-exercises](./js-string-array-exercises)
Exercícios com strings e arrays: encontrar o maior número, ordenar em ordem crescente, verificar palíndromos, checar subconjunto de string e identificar o dia da semana a partir de uma data.

**Tech:** HTML · CSS · JavaScript (String, Array, Date)

---

### 🔤 [text-case-converter](./text-case-converter)
Conversor de texto entre maiúsculas e minúsculas com seleção por rádio button e validação de campo vazio.

**Tech:** HTML · CSS · JavaScript (toUpperCase, toLowerCase)

---

### 🖼️ [interactive-image-states](./interactive-image-states)
Janela interativa que muda de estado (fechada, aberta, quebrada) por eventos de mouse (hover, click), com efeitos sonoros em cada transição.

**Tech:** HTML · CSS · JavaScript (eventos DOM, Audio API)

---

### 🧮 [js-basic-calculators](./js-basic-calculators)
Calculadora de média aritmética de notas e calculadora com as quatro operações matemáticas básicas (soma, subtração, multiplicação, divisão e resto).

**Tech:** HTML · CSS · JavaScript

---

### 🧠 [js-oop-fundamentals](./js-oop-fundamentals)
Demonstrações das formas de criar objetos em JavaScript: literal, `new Object()`, funções construtoras, prototype, classes ES6, herança, campos privados com `#`, `Object.create`, `Object.assign` e `Object.fromEntries`.

**Tech:** HTML · JavaScript (OOP, protótipos, ES6+ classes)

---

### 📐 [js-string-array-exercises](./js-string-array-exercises)
*(ver descrição acima)*

---

### 🟢 [node-js-fundamentals](./node-js-fundamentals)
Exercícios de Node.js: leitura de argumentos da linha de comando, leitura de arquivos (síncrono e assíncrono com `fs`), EventEmitter, servidor HTTP nativo com rotas, e introdução ao Express 5.

**Tech:** Node.js · Express 5 · fs · http · events · nodemon

---

### 📊 [css-grid-news-layout](./css-grid-news-layout)
Layout editorial de notícias sobre séries usando CSS Grid com `grid-template-areas`, imagens, iframes e responsividade.

**Tech:** HTML · CSS Grid

---

### 📱 [responsive-media-queries-demo](./responsive-media-queries-demo)
Demonstração de media queries: div que muda de cor e exibe o nome da cor atual conforme o breakpoint ativo (mobile até 4K).

**Tech:** HTML · CSS (Media Queries, pseudo-elemento `::before`)

---

### 📄 [automated-testing-tools-intro](./automated-testing-tools-intro)
Trabalho escrito sobre ferramentas de teste e introdução a testes automatizados no front-end.

**Tech:** PDF
